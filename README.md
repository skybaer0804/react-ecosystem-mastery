## 🚀 시작하기

1. **환경 준비**
   ```bash
   # Node.js 설치 (18+ 권장)
   # Vite로 새 프로젝트 생성
   npm create vite@latest my-react-app -- --template react
   cd my-react-app
   npm install
   npm run dev
   ```

2. **학습 순서**
   - React 기초 개념부터 시작
   - 환경설정과 도구 익히기
   - 라우팅과 데이터 관리 학습
   - 성능 최적화 기법 적용

## 🛠️ 추천 도구

- **빌드 도구**: Vite (빠른 개발 서버와 빌드)

- **폼**: React Hook Form
- **스타일링**: Tailwind CSS 또는 Styled Components
- **테스트**: Vitest + React Testing Library
